# Romoz

Landing site for **Romoz | رموز**, a brand and digital agency.

Two versions live in the repo:

- `index.html`: primary Arabic site ("Turning your business into symbols").
- `landing/index.html`: newer English variant ("Romooz — Brand & Digital Agency").

## Stack

Static HTML, CSS, vanilla JavaScript. No build step, no framework.

## Run locally

```bash
npx serve .
```

Open `http://localhost:3000`. Arabic version at `/`, English at `/landing/`.

## Status

Active. Real shipped landing page for the Romoz brand.
